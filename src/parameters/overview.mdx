# paremeters.darkMode

The `darkMode` parameter is used to configure the dark mode behavior of the storybook.

You should define it in the `.storybook/preview.tsx` file.

```tsx
// .storybook/preview.tsx
export default {
  parameters: {
    darkMode: {
      ...
    }
  }
}
```
